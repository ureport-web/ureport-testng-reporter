# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| 1.x | Yes |

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Instead, email the maintainers directly or open a private security advisory via GitHub:
**Security → Advisories → New draft security advisory**.

We aim to respond within 72 hours and release a fix within 14 days for confirmed vulnerabilities.
